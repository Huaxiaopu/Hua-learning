- # 文件自动分类工具 📁

  一款简单实用的文件自动分类工具，帮助你快速整理杂乱的文件夹。

  ## 🚀 下载使用

  点击上方 [main.exe](#) 下载工具

  ## 功能说明

  - 自动分类图片/文档/音频等文件
  - 保留原始文件不删除
  - 错误文件移至`垃圾桶`文件夹

  ## 使用步骤

  1. 双击运行程序
  2. 输入要整理的文件夹路径（如`C:\Downloads`）
  3. 查看自动生成的分类文件夹

  ## 系统要求

  - Windows 10/11 系统
  - 需要管理员权限（如需整理系统文件夹）
